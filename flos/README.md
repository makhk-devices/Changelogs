# Flashing Instructions
### Clean Flash (From other ROMs)
- Reboot to recovery
- Flash ROM, Gapps and Magisk (Optional)
- Format DATA (!! Backup data before doing this !!)
- Reboot

### Dirty Flash (From previous build of same ROM)
- Reboot to recovery
- Flash ROM, Gapps and Magisk (Optional)
- Wipe Caches
- Reboot
